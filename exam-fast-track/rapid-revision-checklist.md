# Rapid Revision Checklist

## Daily Pre-Session (5 min)

- [ ] Correct context selected
- [ ] Namespace set
- [ ] Aliases loaded
- [ ] One objective defined for this session

## Session Execution (45-120 min)

- [ ] Study one focused topic
- [ ] Execute one real lab
- [ ] Save one working YAML in topic `examples/`
- [ ] Record one mistake + fix

## Session Close (5-10 min)

- [ ] Verify notes are short and exam-useful
- [ ] Promote reusable YAML to `templates/` if generic
- [ ] Update weak areas in mistakes logs
- [ ] Decide next session topic

## Last Week Rule

Do not collect more theory. Convert everything into commands, YAML, and troubleshooting moves.
