# YAML Templates

Reusable Kubernetes manifests for exam speed and daily practice.

## Usage

1. Copy a template.
2. Rename resources and namespace.
3. Apply and verify.
4. Keep final validated version in topic `examples/`.

## Starter Set

- pod, deployment, service
- configmap, secret
- pvc, statefulset
- role, rolebinding
- networkpolicy
- job, cronjob
- ingress, hpa, pdb
